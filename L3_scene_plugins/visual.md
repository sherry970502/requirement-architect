# L3 场景插件 · 视觉作品类

适用:画册、插画集、海报、品牌视觉、绘本、展览视觉、书籍装帧等。

## 形态维度清单

当用户在 L1-L2 完成选择后,针对视觉作品类项目,按以下维度给出推荐默认值:

### 1. 载体形式
- 纸质为主
- 电子为主
- 双载体并行

**推荐规则:**
- 核心价值偏"作品价值/品牌价值" → 纸质为主
- 核心价值偏"商业价值"且面向年轻受众 → 电子为主(或双载体)
- 目标用户含"收藏品路径"或"国风路径" → 纸质为主
- 目标用户含"启蒙路径"且学龄前 → 纸质为主(耐用性)

### 2. 规模/体量
- 小品级(1-10 件/页)
- 标准级(10-40 件/页)
- 大部头(40+ 件/页)

**推荐规则:**
- 作品集类 → 标准级(40-60 页典型)
- 启蒙/收藏品 → 标准级
- 一次性海报/插画 → 小品级
- 大型主题项目 → 大部头

### 3. 完成度标准
- 作品集标准(每一件都要能单拿出来)
- 合集标准(整体成立,单件可有高低)
- 实验标准(探索方向,不追求每件完成度)

**推荐规则:**
- 作品价值驱动 → 作品集标准
- 商业价值驱动 → 作品集标准
- 品牌价值驱动 → 合集标准或作品集标准
- 精神价值驱动 → 看用户状态,可以是实验标准

### 4. 出版/发行意图
- 先按可出版标准做,做完再定
- 明确不出版,自留或小范围分享
- 以出版/投稿为目标

**推荐规则(默认值):** "先按可出版标准做,做完再定"——除非用户明确说过意图。这是作品类项目最安全的默认。

### 5. 制作周期
- 短周期(1-3 个月)
- 中周期(3-6 个月)
- 长周期(6 个月以上)

**推荐规则:** 基于规模和完成度标准推算。用户如有明确时间压力会在 L1 阶段提到。

---

## 需要高亮让用户关注的维度

每次输出形态建议时,以下几项**即使给了默认推荐,也要标 ⚠️ 让用户确认**:

- **载体形式**(纸质 vs 电子,差别巨大,影响一切后续决策)
- **出版意图**(是否追求出版,会影响完成度标准和规格)

其他维度默认给推荐即可(✅),用户不反对就通过。

## L4 规格层自动推导规则

基于 L1-L3 的结论,自动推出以下参数(写入需求简报,不问用户):

### 画册类
- **尺寸**:
  - 作品集标准 → 16 开(210×285mm)或 大 16 开(225×300mm)
  - 收藏品/国风 → 可考虑特殊尺寸(方形、瘦长)
  - 启蒙/儿童 → 近方形,便于低龄阅读
- **装订**:
  - 高端作品集 → 精装锁线,可考虑护封
  - 常规画册 → 平装胶订
- **用纸**:
  - 艺术性强 → 特种纸(哑粉、艺术纸、棉质纸)
  - 商业出版 → 铜版纸/亚粉纸
- **工艺**:
  - 高端 → 可考虑烫金、压纹、UV 局部光
  - 常规 → 覆哑膜/亮膜

### 海报/插画类
- **尺寸**:A3 / A2 / 70×100cm / 根据展陈场景定
- **分辨率**:300dpi(印刷)/ 150dpi(大幅展陈)
- **色彩模式**:CMYK(印刷)/ RGB(电子展示)

### 品牌视觉类
- **尺寸**:按使用场景输出多版本(LOGO 横版/竖版/单色版等)
- **格式**:AI/SVG 源文件 + PNG/JPG 导出版

### 电子版本通用
- **分辨率**:屏幕阅读 150-200ppi 即可
- **格式**:PDF(跨端通用)/ EPUB(如有文字阅读需求)
