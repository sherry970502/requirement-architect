# L3 场景插件 · 内容产品类

适用:课程、文章、视频、专栏、书籍、IP、Newsletter、Podcast、教程等。

## 形态维度清单

### 1. 内容载体
- 图文(文章、推送、长文)
- 视频(短视频、长视频、课程视频)
- 音频(Podcast、课程音频)
- 混合(图文+视频+音频)

**推荐规则:**
- 目标用户偏知识消费 → 图文 或 视频
- 目标用户偏碎片场景 → 短视频 或 音频
- 深度内容 + 专业受众 → 图文长文
- 教学性强 → 视频课程

### 2. 发布结构
- 单篇/单期(一次性完成)
- 系列(有限集,3-12 期)
- 连载(持续更新,无固定期数)
- 模块化(主题独立,可按需组合)

**推荐规则:**
- 项目性 → 单篇/单期 或 系列
- 长期经营 → 连载 或 模块化
- 课程类 → 系列 或 模块化

### 3. 时长/篇幅
- 极短(短视频 < 3 分钟 / 推送 < 800 字)
- 标准(视频 5-15 分钟 / 文章 1500-3000 字 / 课程单节 10-20 分钟)
- 深度(视频 20+ 分钟 / 长文 5000+ 字 / 深度课程)

**推荐规则:**
- 使用场景"碎片化" → 极短
- 使用场景"专注学习" → 标准 或 深度
- 使用场景"陪伴背景" → 标准(音频尤其)

### 4. 互动设计
- 单向输出(观众只接收)
- 引导参与(评论、反馈、小作业)
- 社群联动(有配套社群、答疑)

**推荐规则:**
- 一次性作品 → 单向输出
- 持续经营 + 商业化意图 → 引导参与 或 社群联动
- 收费课程 → 社群联动

### 5. 发布平台
- 自有平台(公众号、个人站、Newsletter)
- 第三方平台(知乎、小红书、B 站、YouTube、播客平台)
- 分发矩阵(多平台同步)

**推荐规则:**
- 品牌价值驱动 + 长期 → 自有平台 + 分发矩阵
- 商业价值驱动 → 根据目标用户在哪选择主平台
- 作品价值驱动 → 自有平台为主

---

## 需要高亮让用户关注的维度

- **发布结构**(决定工作量是一次性还是长期投入,差别巨大)
- **发布平台**(不同平台的内容形态完全不同,不能通用)

## L4 规格层自动推导规则

### 图文内容
- **篇幅**:按上面"时长/篇幅"建议
- **结构**:
  - 标题 + 引言(钩子)+ 主体(2-5 个核心要点)+ 结尾
  - 每 300-500 字一个视觉断点(小标题、配图、引用)
- **配图**:每篇 1-5 张(视平台)

### 视频内容
- **分辨率**:1080p(通用)/ 4K(高端项目)
- **时长**:按上面建议
- **结构**:
  - 短视频:3 秒钩子 + 主体 + 15 秒内完成核心信息
  - 长视频:前 30 秒建立价值预期 + 章节化结构
- **辅助**:字幕(必备)、封面图(必备)

### 音频内容
- **时长**:单集 20-50 分钟典型
- **结构**:片头(15-30 秒)+ 内容 + 片尾
- **规格**:立体声,128-192 kbps MP3

### 课程类
- **整体规模**:10-30 节典型(根据模块化程度)
- **单节**:10-20 分钟
- **配套**:课程大纲 + 每节简介 + 作业/练习(视需要)

### 持续经营类(连载/Newsletter)
- **更新频率**:周更 / 双周更 / 月更(视精力和目标用户习惯)
- **预设储备**:首发前准备 3-5 期存稿,避免断更
